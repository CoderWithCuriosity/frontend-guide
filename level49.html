DOM Manipulation

DOM manipulation allows you to change HTML elements dynamically. You can change text content, styles, attributes, and more.

textContent
Change text content
element.textContent = "New text"
style
Change CSS styles
element.style.color = "red"
setAttribute
Change attributes
element.setAttribute("id", "newId")

// DOM Manipulation Examples
> let element = document.getElementById("demoText");

// Change text content
> element.textContent = "New text content!";
> console.log(element.textContent);
"New text content!"

// Change style
> element.style.color = "#f0c674";
> element.style.fontSize = "1.5rem";

// Change attribute
> element.setAttribute("data-type", "modified");
> console.log(element.getAttribute("data-type"));
"modified"

Key Takeaways

  • textContent - Gets or sets the text content of an element
  • style - Accesses inline styles of an element
  • setAttribute() - Sets an attribute on an element
  • getAttribute() - Gets the value of an attribute
  • classList - Adds, removes, or toggles CSS classes
  • innerHTML - Gets or sets HTML content (use with caution)
