Level 3: Text Formatting Tags

GOAL: Build a webpage demonstrating various HTML text formatting tags

Skills You'll Use

  • HTML: html, head, title, body, h1, h2, p, strong, i, em, u, mark, small, del, ins, sub, sup, br, hr
  • CSS: No CSS used in this level
  • JS: No JavaScript used in this level

Example Preview

Build It Step-by-Step

  1. Create the basic HTML structure with html, head, title, and body tags
  2. Add a main heading using h1 tag
  3. Create sections with h2 subheadings for different formatting categories
  4. Use p tags for paragraphs with various formatting tags inside
  5. Apply strong for bold text, em for italic text, and u for underlined text
  6. Use mark for highlighted text, small for smaller text
  7. Implement del for deleted text and ins for inserted text
  8. Add sub for subscript and sup for superscript text
  9. Use br for line breaks and hr for horizontal rules between sections
