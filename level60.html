setTimeout & setInterval_

Learn how to schedule code execution with setTimeout and setInterval. Run code after a delay or repeatedly at intervals.

JavaScript Level 60 Quiz Required

setTimeout and setInterval

setTimeout and setInterval are used to schedule code execution. setTimeout runs once after a delay, setInterval runs repeatedly.

Try It Yourself

Timer 0
// Click a button to start

Syntax

// setTimeout - runs once after delay
setTimeout(() => {
  console.log("Runs after 1 second");
}, 1000);

// setInterval - runs repeatedly
setInterval(() => {
  console.log("Runs every second");
}, 1000);

// Clear interval
clearInterval(intervalId);
clearTimeout(timeoutId);

Console Output

// setTimeout Example
> setTimeout(() => {
> console.log("Delayed message");
> }, 2000);

// setInterval Example
> let count = 0;
> let interval = setInterval(() => {
> count++;
> console.log("Count:", count);
> if (count === 5) {
> clearInterval(interval);
> }
> }, 1000);

Key Takeaways

  • setTimeout() - Runs code once after a delay (in milliseconds)
  • setInterval() - Runs code repeatedly at intervals
  • clearTimeout() - Cancels a scheduled timeout
  • clearInterval() - Stops an interval
  • Return value - Both return an ID that can be used to cancel
  • Milliseconds - 1000ms = 1 second

Your Challenge

Create a new file named level[level].html. Inside it, add a <script> tag and place your solution for this level within the script.

Practice setTimeout and setInterval by doing the following:

  • setTimeout Delay: Use setTimeout() to log "Hello after 3 seconds!" to the console after a 3-second delay. Syntax: setTimeout(function() { console.log("Hello after 3 seconds!"); }, 3000). The time is in milliseconds (3000ms = 3 seconds)
  • setInterval Counter: Use setInterval() to create a counter that starts at 1 and counts up to 10. Create a variable count = 1 and use setInterval to log the current count and increment it each time. Stop the interval when count reaches 10 using clearInterval()
  • Stop Interval After 5 Seconds: Start a setInterval that logs "Tick!" every second. Use setTimeout to stop the interval after exactly 5 seconds using clearInterval(). This shows how to stop an interval from outside
  • Countdown Timer: Create a countdown timer that starts from 10 and counts down to 0. Use setInterval to decrement the counter each second. When it reaches 0, log "Blast off!" and clear the interval. Display each number in the console as it counts down
  • Combine setTimeout and setInterval: Start a setInterval that logs "Still running..." every 2 seconds. But also use setTimeout to wait 6 seconds, then log "6 seconds passed! Now stopping..." and clearInterval to stop the interval. This combines both timing methods

💡 Hint: setTimeout(callback, milliseconds) runs once after the delay. setInterval(callback, milliseconds) runs repeatedly every interval. Both return an ID that you can use with clearTimeout() or clearInterval() to stop them. Remember: 1000ms = 1 second. Use clearInterval() inside your interval when you want it to stop automatically!
