Level 6: Images and Alt Text

GOAL: Build a webpage demonstrating HTML image tags with proper alt text

Skills You'll Use

  • HTML: html, head, title, body, h1, h2, p, img, ul, li, br, hr
  • CSS: No CSS used in this level
  • JS: No JavaScript used in this level

Example Preview

Build It Step-by-Step

  1. Create the basic HTML structure with html, head, title, and body tags
  2. Add a main heading using h1 tag
  3. Create sections with h2 subheadings for different image examples
  4. Use img tags with src attribute to display images
  5. Implement alt attribute for accessibility and broken images
  6. Use width and height attributes to control image dimensions
  7. Create image links by wrapping img tags with a tags
  8. Use ul and li for organizing content
  9. Apply br for line breaks and hr for horizontal rules between sections
