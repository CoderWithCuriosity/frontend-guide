Level 8: CSS Backgrounds & Borders

GOAL: Build a webpage demonstrating CSS background and border properties

Skills You'll Use

  • HTML: html, head, title, style, body, h1, h2, p, div, ul, li, br, hr
  • CSS: background-color, background-image, border, border-radius, padding, margin, width, height
  • JS: No JavaScript used in this level

Example Preview

Build It Step-by-Step

  1. Create the basic HTML structure with html, head, title, and body tags
  2. Add a style tag in the head section for CSS
  3. Use background-color with different color values
  4. Apply background-image with linear gradients
  5. Implement border property with different styles, widths, and colors
  6. Use border-radius to create rounded corners
  7. Apply padding and margin for spacing
  8. Use width and height to control element dimensions
  9. Combine multiple background and border properties
