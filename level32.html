Variables in JavaScript

Variables are containers for storing data values. In JavaScript, we use let and const to declare variables.

let Variables

  • Value can be changed
  • Use when the value might update
  • Block-scoped
  • Example: let age = 25;

const Variables

  • Value cannot be changed
  • Use for constants
  • Block-scoped
  • Example: const PI = 3.14159;

let Variables (Can Change)

let name = "John";
let age = 25;
let city = "New York";

These can be changed later

const Variables (Cannot Change)

const PI = 3.14159;
const BIRTH_YEAR = 1998;
const COUNTRY = "USA";

These cannot be changed

Console Output

// Console Output
> console.log(name);
John
> console.log(age);
25
> console.log(PI);
3.14159
> console.log(BIRTH_YEAR);
1998
