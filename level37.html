Comparison Operators

Comparison operators compare two values and return true or false. They are commonly used in if statements, loops, and conditions.

===
Equal (value & type)
5 === 5
true
!==
Not Equal (value & type)
5 !== 3
true
>
Greater Than
5 > 3
true
<
Less Than
3 < 5
true

Comparison Operators Reference

Operator Description Example (x=5, y=3) Result
=== Equal value and type 5 === 5 true
!== Not equal value or type 5 !== 3 true
> Greater than 5 > 3 true
< Less than 3 < 5 true
>= Greater than or equal 5 >= 5 true
<= Less than or equal 3 <= 5 true

Key Takeaways

  • === - Strict equality: checks both value and type
  • !== - Strict inequality: checks both value and type
  • > - Greater than
  • < - Less than
  • >= - Greater than or equal to
  • <= - Less than or equal to
  • Always use === and !== instead of == and !=
  • Comparisons always return a boolean (true or false)
