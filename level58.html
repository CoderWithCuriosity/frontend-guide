Async/Await

Async/await is a modern way to handle asynchronous operations. It makes your code look synchronous and easier to understand.

Syntax

// Async function declaration
async function fetchData() {
  try {
    let response = await fetch("url");
    let data = await response.json();
    return data;
  } catch (error) {
    console.log(error);
  }
}

Key Takeaways

  • async - Declares an asynchronous function
  • await - Waits for a Promise to resolve
  • try/catch - Handles errors in async functions
  • Readability - Async/await looks like synchronous code
  • Parallel - Use Promise.all() for parallel requests
  • Returns Promise - Async functions always return a Promise
