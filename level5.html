Level 5: Links and Anchors

GOAL: Build a webpage demonstrating HTML links and anchor tags

Skills You'll Use

  • HTML: html, head, title, body, h1, h2, p, a, ul, li, br, hr
  • CSS: No CSS used in this level
  • JS: No JavaScript used in this level

Example Preview

Build It Step-by-Step

  1. Create the basic HTML structure with html, head, title, and body tags
  2. Add a main heading using h1 tag
  3. Create sections with h2 subheadings for different link types
  4. Use a tags with href attribute for external links
  5. Create internal page anchors using id attributes and a tags with href="#id"
  6. Implement email links using mailto: in href attribute
  7. Use ul and li for organizing link lists
  8. Apply br for line breaks and hr for horizontal rules between sections
