Template Literals

Template literals are strings that allow embedded expressions and multi-line text. They use backticks (`) instead of quotes and ${} for expressions.

Try It Yourself

Click "Generate" to see template literal output...

Syntax

// Basic template literal
let name = "John";
let greeting = `Hello, ${name}!`;

// Multi-line string
let message = `This is a
multi-line
string`;

// Expressions
let result = `2 + 3 = ${2 + 3}`;

// Function calls
let info = `Hello ${name.toUpperCase()}!`;

Console Output

// Template Literal Examples
> let name = "John";
> let age = 25;
> let greeting = `Hello, ${name}! You are ${age} years old.`;
> console.log(greeting);
Hello, John! You are 25 years old.

// Expression evaluation
> let result = `${5 + 3} is the sum`;
> console.log(result);
8 is the sum

// Multi-line string
> let multi = `Line 1
Line 2
Line 3`;
> console.log(multi);
Line 1
Line 2
Line 3

Key Takeaways

  • Backticks (`) - Used instead of quotes
  • ${expression} - Embeds expressions in strings
  • Multi-line - Preserves line breaks without \n
  • Expressions - Can include variables, math, function calls
  • Readability - Easier to read than concatenation
  • Tagged templates - Advanced feature for custom processing

Your Challenge

Create a new file named level[level].html. Inside it, add a <script> tag and place your solution for this level within the script.

Practice template literals by doing the following:

  • Template with Multiple Variables: Create variables like let firstName = "John", let lastName = "Doe", and let age = 25. Use a template literal with backticks to create a sentence like `My name is ${firstName} ${lastName} and I am ${age} years old.`. Log the result to the console
  • Math Expression Inside ${}: Create variables like let x = 10 and let y = 5. Use a template literal that does math inside ${} like `${x} + ${y} = ${x + y}` and `${x} * ${y} = ${x * y}`. Also try `The average of ${x} and ${y} is ${(x + y) / 2}`
  • Multi-line String with Backticks: Create a multi-line string using backticks (not quotes!). Example: let poem = `Roses are red, Violets are blue, JavaScript is fun, And so are you!`. Notice that line breaks are preserved without needing \n. Log the poem to see it formatted across multiple lines
  • Function Call Inside Template: Create a function called getGreeting() that returns "Good morning!". Then create another function called getTimeOfDay() that returns "evening" or "morning". Use template literals to call functions inside ${} like `Hello! ${getGreeting()} It's a beautiful ${getTimeOfDay()}!`. Log the result
  • HTML Template with Template Literals: Create variables like let userName = "Alice", let userAge = 28, and let userCity = "New York". Use a template literal to create HTML like let html = `<div class="user-card"><h2>${userName}</h2><p>Age: ${userAge}</p><p>City: ${userCity}</p></div>`. Log the HTML string and notice how it looks like real HTML markup

💡 Hint: Template literals use backticks (`) instead of quotes. Place expressions inside ${} - they can be variables, math operations, or even function calls! Multi-line strings are easy with backticks (just press Enter). Template literals are great for creating HTML, messages, and dynamic strings. Always use console.log() to see your results!
