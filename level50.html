Event Listeners

Event listeners let you execute code when a user interacts with your page. Common events include clicks, mouse movements, keyboard presses, and more.

click
When element is clicked
element.addEventListener("click", function)
mouseover/mouseout
When mouse enters/leaves
element.addEventListener("mouseover", function)
keydown/keyup
When keys are pressed
document.addEventListener("keydown", function)

Syntax

// Adding an event listener
element.addEventListener("click", function() {
  // code to run when clicked
});

// Using a named function
function handleClick() {
  console.log("Clicked!");
}
element.addEventListener("click", handleClick);

// Removing an event listener
element.removeEventListener("click", handleClick);

Key Takeaways

  • addEventListener() - Attaches an event to an element
  • removeEventListener() - Removes an event from an element
  • click - Fires when an element is clicked
  • mouseover/mouseout - Fires when mouse enters/leaves
  • keydown/keyup - Fires when keys are pressed/released
  • Event object - Provides information about the event
