Text Alignment & Spacing Demonstration

Text Alignment

Left Aligned (Default)

This text is aligned to the left side of the container. This is the default alignment for most languages that read left-to-right.

Center Aligned

This text is centered within its container. Center alignment is often used for headings, titles, and special emphasis.

Right Aligned

This text is aligned to the right side of the container. Right alignment is common for numbers and dates in tables.

Justified Alignment

This text is justified, meaning it stretches to fill the entire width of the container. Justified text creates clean edges on both sides but can sometimes create uneven spacing between words. This is commonly used in newspapers, books, and magazines for a formal appearance.

Line Height

Normal Line Height

This paragraph has normal line height. The spacing between lines is determined by the browser's default settings. This text demonstrates how normal line spacing appears in a typical paragraph with multiple lines to show the vertical rhythm of the text.

Tight Line Height (1.2)

This paragraph has tight line height (1.2 times the font size). The lines are closer together which can be useful for saving space. However, text that is too tightly spaced can become difficult to read, especially with longer paragraphs or smaller font sizes.

Loose Line Height (2.0)

This paragraph has loose line height (2.0 times the font size). The increased spacing between lines improves readability and gives a more open, airy feel to the text. This is often used in designs that prioritize readability and accessibility.

Text Indentation

No Indentation

This paragraph has no first line indentation. All lines start at the same position, which is common in web content and digital media where space is limited and readability is prioritized.

Standard Indentation (40px)

This paragraph has a standard first line indentation of 40 pixels. The first line is indented while subsequent lines start at the normal margin. This traditional formatting style is common in printed books and formal documents.

Negative Indentation (-20px)

This paragraph has a negative first line indentation of -20 pixels with a compensating left margin. The first line extends to the left of the normal text block, creating a hanging indent effect often used in bibliographies and reference lists.

Letter & Word Spacing

Normal Spacing

This text has normal letter and word spacing. The distance between characters and words follows the font's default metrics, providing optimal readability for continuous text.

Wide Letter Spacing (3px)

This text has increased letter spacing of 3 pixels. Wider letter spacing can improve readability for headings, logos, or text in all caps, but may reduce reading speed for body text.

Tight Letter Spacing (-1px)

This text has reduced letter spacing of -1 pixel. Tighter letter spacing can create a denser, more compact appearance but may reduce legibility if overused.

Wide Word Spacing (10px)

This text has increased word spacing of 10 pixels. Wider word spacing creates more separation between words which can sometimes improve readability in justified text or specific design contexts.

Tight Word Spacing (-2px)

This text has reduced word spacing of -2 pixels. Tighter word spacing brings words closer together, which can be useful for fitting more text in limited spaces but may reduce readability.
