Level 7: Basic CSS Colors & Fonts

GOAL: Build a webpage demonstrating basic CSS colors and font properties

Skills You'll Use

  • HTML: html, head, title, style, body, h1, h2, p, div, span, ul, li, br, hr
  • CSS: color, background-color, font-family, font-size, font-weight, font-style, text-align
  • JS: No JavaScript used in this level

Example Preview

Build It Step-by-Step

  1. Create the basic HTML structure with html, head, title, and body tags
  2. Add a style tag in the head section for CSS
  3. Use color property to change text colors
  4. Apply background-color property for element backgrounds
  5. Use font-family to change typefaces
  6. Implement font-size to control text size
  7. Apply font-weight for bold text and font-style for italic
  8. Use text-align for text alignment
  9. Use div and span for applying styles to specific sections
