Level 10: Simple Div Layout

GOAL: Build a webpage demonstrating simple div-based layout structure

Skills You'll Use

  • HTML: html, head, title, style, body, h1, h2, h3, p, div, span, ul, li, br, hr
  • CSS: width, height, margin, padding, background-color, border, float, clear, text-align
  • JS: No JavaScript used in this level

Example Preview

Build It Step-by-Step

  1. Create the basic HTML structure with html, head, title, and body tags
  2. Add a style tag in the head section for CSS
  3. Use div elements to create layout containers
  4. Apply width and height to control container dimensions
  5. Use margin and padding for spacing between elements
  6. Implement background-color to distinguish different sections
  7. Apply float property for basic column layouts
  8. Use clear property to manage float behavior
  9. Create header, content, sidebar, and footer sections
