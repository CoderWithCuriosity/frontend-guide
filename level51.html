Key Takeaways

  • value - Gets or sets the value of an input
  • input event - Fires when the value changes
  • submit event - Fires when a form is submitted
  • preventDefault() - Prevents default form submission
  • type - Different input types: text, number, email, password
  • placeholder - Shows hint text in the input
