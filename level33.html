JavaScript Data Types

JavaScript has several data types that determine how values are stored and manipulated. Understanding these types is essential for writing correct and efficient code.

String
"Hello World"
typeof: "string"
Number
42
typeof: "number"
Number (Decimal)
3.14159
typeof: "number"
Boolean
true
typeof: "boolean"
Undefined
undefined
typeof: "undefined"
Null
null
typeof: "object"

"number"

> typeof null

Data Types Reference

Type Description Example
String Text data, wrapped in quotes "Hello", 'World'
Number Numeric data (integers and decimals) 42, 3.14, -5
Boolean Logical true or false true, false
Undefined Variable exists but has no value undefined
Null Intentionally empty value null

Key Takeaways

  • String - Text data, always in quotes
  • Number - Both integers and decimals
  • Boolean - Only true or false
  • Undefined - Variable exists but has no value
  • Null - Intentionally empty value
  • Use typeof to check data types
