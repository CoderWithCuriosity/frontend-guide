JSON Basics

JSON (JavaScript Object Notation) is a lightweight data format for exchanging information. It's text-based and easy to read, making it perfect for APIs and data storage.

JSON.stringify()
Convert object to JSON string
JSON.stringify({ name: "John" })
JSON.parse()
Convert JSON string to object
JSON.parse('{"name":"John"}')

Console Output

// JSON Examples
> let obj = { name: "John", age: 25, city: "New York" };
> let jsonString = JSON.stringify(obj);
> console.log(jsonString);
{"name":"John","age":25,"city":"New York"}

> let parsed = JSON.parse(jsonString);
> console.log(parsed.name);
"John"

// Arrays in JSON
> let arr = ["apple", "banana", "orange"];
> let arrJson = JSON.stringify(arr);
> console.log(arrJson);
["apple","banana","orange"]

Key Takeaways

  • JSON.stringify() - Converts objects to JSON strings
  • JSON.parse() - Converts JSON strings back to objects
  • JSON is text - Always a string when transmitted
  • Data types - Supports strings, numbers, booleans, arrays, objects, null
  • Keys must be strings - Always use double quotes for keys
  • No functions - JSON cannot store functions or undefined
