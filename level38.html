Logical Operators

Key Takeaways

  • && (AND) - Returns true only if both conditions are true
  • || (OR) - Returns true if at least one condition is true
  • ! (NOT) - Reverses the boolean value
  • AND has higher precedence than OR
  • Use parentheses to control evaluation order: (a && b) || c
  • Logical operators are often used with comparison operators

Practice logical operators by doing the following:

  • AND Operator (&&): Create a variable num and check if it's between 10 and 20 using num >= 10 && num <= 20
  • OR Operator (||): With the same num, check if it's less than 10 OR greater than 20 using num < 10 || num > 20
  • NOT Operator (!): Create a variable value and use ! to check if it's NOT null (like value !== null or just !value)
  • Combine Multiple: Create variables like age, hasLicense, and isStudent, then combine operators: (age >= 18 && hasLicense) || isStudent
  • Complex If Statements: Write an if statement with multiple conditions like checking if someone can get a discount: if (age < 12 || age >= 65 || isStudent)

💡 Hint: Use console.log() to see if your conditions return true or false!
