Level 4: Lists (Ordered & Unordered)

GOAL: Build a webpage demonstrating HTML ordered and unordered lists

Skills You'll Use

  • HTML: html, head, title, body, h1, h2, p, ul, ol, li, dl, dt, dd, br, hr
  • CSS: No CSS used in this level
  • JS: No JavaScript used in this level

Example Preview

Build It Step-by-Step

  1. Create the basic HTML structure with html, head, title, and body tags
  2. Add a main heading using h1 tag
  3. Create sections with h2 subheadings for different list types
  4. Use ul for unordered lists with li for list items
  5. Use ol for ordered lists with li for list items
  6. Create nested lists by placing ul or ol inside li elements
  7. Implement dl for definition lists with dt for terms and dd for definitions
  8. Use br for line breaks and hr for horizontal rules between sections
