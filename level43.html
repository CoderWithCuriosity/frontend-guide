Functions with Parameters

Parameters allow you to pass data into functions, making them more flexible and reusable. You can use different values each time you call a function.

One Parameter
Pass a single value
greet("John")
Multiple Parameters
Pass several values
add(5, 3)
Different Types
String, number, boolean
info("John", 25, true)

Syntax

// Function with one parameter
function functionName(parameter) {
  console.log(parameter);
}

functionName("value");

// Function with multiple parameters
function add(a, b) {
  return a + b;
}

add(5, 3);

Console Output

// Function with one parameter
> function greet(name) {
> console.log("Hello", name);
> }

> greet("John");
"Hello John"
> greet("Alice");
"Hello Alice"

// Function with multiple parameters
> function add(a, b) {
> console.log(a + b);
> }

> add(5, 3);
8
> add(10, 20);
30

Key Takeaways

  • Parameters - Variables listed in the function definition
  • Arguments - Values passed to the function when called
  • Flexibility - Functions work with different values
  • Reusability - One function, many uses
  • Multiple parameters - Separate with commas
  • Different types - Parameters can be strings, numbers, booleans, etc.

Your Challenge

Create a new file named level[level].html. Inside it, add a <script> tag and place your solution for this level within the script.

Practice function parameters by doing the following:

  • Greet Someone: Create a function called greet() that takes one parameter name and logs "Hello, [name]!" to the console. Call it with "John" and "Sarah"
  • Add Two Numbers: Create a function called add() that takes two parameters a and b, adds them, and logs the result. Call it with add(5, 3) and add(10, 7)
  • Multiply Two Numbers: Create a function called multiply() that takes two parameters x and y, multiplies them, and logs the result. Call it with multiply(4, 5) and multiply(2, 8)
  • Name and Age: Create a function called personInfo() that takes two parameters name and age and logs "[name] is [age] years old". Call it with personInfo("Alice", 25) and personInfo("Bob", 30)
  • Reuse with Different Values: Call all your functions multiple times with different values (like different names, different numbers) to see how parameters make functions flexible and reusable

💡 Hint: Parameters are like placeholders in your function: function functionName(parameter1, parameter2) { }. When you call the function, you pass in actual values (arguments): functionName("value1", 5)
