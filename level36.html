if/else Statements

if/else statements allow your code to make decisions. Based on whether a condition is true or false, different blocks of code will execute.

if (true)
if (age >= 18)
"You are an adult"
if (false)
if (age < 18)
"You are a minor"
if...else
if (score >= 60)
"Pass" else "Fail"
Nested if
if (age >= 18) {
if (hasLicense)

if/else Syntax Reference

Syntax Description Example
if (condition) Runs code if condition is true if (x > 5) { ... }
else Runs code if condition is false else { ... }
else if Checks another condition else if (y < 10) { ... }
Nested if if inside another if if (x > 5) { if (y > 3) { ... } }

Console Output

// if/else Statements Console Output

// More examples
> let score = 85;
> if (score >= 90) {
"A"
> } else if (score >= 80) {
"B"
> } else {
"C"
> }

> Result: "B"

Key Takeaways

  • if - Executes code when condition is true
  • else - Executes code when condition is false
  • else if - Checks additional conditions
  • Nested if - if statements inside other if statements
  • Conditions use comparison operators: >, <, >=, <=, ===, !==
  • Use { } to group multiple statements
