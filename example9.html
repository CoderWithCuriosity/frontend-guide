Understanding the Box Model

Content Area

Green: Content | Blue: Padding | Red: Border | Outside: Margin

Padding Examples

Padding: 10px (all sides)
Padding: 20px (top/bottom) 10px (left/right)
Padding: 10px(top) 20px(right) 30px(bottom) 40px(left)

Box-Sizing Property

content-box
Width: 200px + padding + border
Total: 200 + 40 + 10 = 250px
border-box
Width: 200px includes padding & border
Content: 200 - 40 - 10 = 150px

Practical Example

Card Component

This card demonstrates box model properties:

  • Margin: Creates space around the card
  • Border: Defines the card boundary
  • Padding: Creates space inside the card
  • Content: The actual text and elements
Total width = margin + border + padding + content
