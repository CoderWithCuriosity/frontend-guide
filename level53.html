Local Storage

localStorage allows you to store data in the browser that persists even after the page is closed and reopened. It's perfect for saving user preferences, settings, and application state.

setItem()
Store data
localStorage.setItem("key", "value")
getItem()
Retrieve data
localStorage.getItem("key")
removeItem()
Delete data
localStorage.removeItem("key")

Console Output

// Local Storage Examples
> localStorage.setItem("username", "JohnDoe");
> let name = localStorage.getItem("username");
> console.log(name);
"JohnDoe"

> localStorage.removeItem("username");
> console.log(localStorage.getItem("username"));
null

// Storing objects (JSON)
> let user = { name: "John", age: 25 };
> localStorage.setItem("user", JSON.stringify(user));
> let stored = JSON.parse(localStorage.getItem("user"));
> console.log(stored.name);
"John"

Key Takeaways

  • setItem(key, value) - Saves data to localStorage
  • getItem(key) - Retrieves data from localStorage
  • removeItem(key) - Deletes data from localStorage
  • clear() - Deletes all data from localStorage
  • JSON.stringify() - Converts objects to strings
  • JSON.parse() - Converts strings back to objects

Your Challenge

Create a new file named level[level].html. Inside it, add a <script> tag and place your solution for this level within the script.

Practice localStorage by doing the following:

  • Save Name: Create a variable userName with your name (like "John") and save it to localStorage using localStorage.setItem("name", userName). Open your browser's Application tab or Console to see it saved
  • Retrieve and Display: Retrieve your name from localStorage using localStorage.getItem("name") and store it in a variable like let savedName = localStorage.getItem("name"). Then log it to the console using console.log("Saved name: " + savedName)
  • Save Object with JSON: Create an object like let person = { name: "Alice", age: 25, city: "New York" }. Convert it to a JSON string using JSON.stringify(person) and save it to localStorage with a key like "personData". Example: localStorage.setItem("personData", JSON.stringify(person))
  • Remove Specific Key: Remove your name from localStorage using localStorage.removeItem("name"). Then try retrieving it again with localStorage.getItem("name") and log it to see that it's now null
  • Clear All Data: Use localStorage.clear() to remove ALL saved data from localStorage. Check if it's empty by logging the length: console.log(localStorage.length) which should show 0

💡 Hint: localStorage saves data as strings. Use JSON.stringify() to convert objects to strings before saving. Use JSON.parse() to convert strings back to objects when retrieving. Check the Application tab in DevTools (F12) to see all stored data. Remember: setItem() saves, getItem() retrieves, removeItem() deletes one key, and clear() deletes everything!
