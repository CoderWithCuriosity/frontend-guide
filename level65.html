Classes Basics

Classes are blueprints for creating objects. They define properties and methods that all instances of the class will have.

Try It Yourself

Click a button to see classes in action...

Syntax

// Class definition
class Person {
  constructor(name, age) {
    this.name = name;
    this.age = age;
  }

  greet() {
    return `Hello, ${this.name}`;
  }
}

// Creating an instance
let john = new Person("John", 25);
console.log(john.greet()); // "Hello, John"

Key Takeaways

  • class - Keyword to define a class
  • constructor - Special method for initializing objects
  • this - Refers to the current instance
  • new - Keyword to create instances
  • Methods - Functions defined inside the class
  • Properties - Values stored on the instance

Your Challenge

Practice classes by doing the following:

  • Create Book Class: Create a class called Book with a constructor that takes three parameters: title, author, and year. Inside the constructor, assign these parameters to properties using this.title = title, this.author = author, and this.year = year
  • Add Method for Book Info: Inside the Book class, add a method called getInfo() that returns a string with the book information. Example: return "${this.title} by ${this.author} (${this.year})". Test it by creating a book and calling the method
  • Create Multiple Book Instances: Create at least three different book objects using the Book class. Example: let book1 = new Book("The Hobbit", "J.R.R. Tolkien", 1937), let book2 = new Book("1984", "George Orwell", 1949), and let book3 = new Book("Dune", "Frank Herbert", 1965). Log each book to the console
  • Add Method to Check if Book is Old: Inside the Book class, add a method called isOld() that returns true if the book was published more than 50 years ago, and false otherwise. Use the current year: let currentYear = new Date().getFullYear(); return (currentYear - this.year) > 50. Test it on your book instances
  • Create a Class that Extends Another: Create a class called EBook that extends the Book class. Add a new property called fileSize in the constructor. Use super(title, author, year) to call the parent constructor. Add a method called getFileInfo() that returns "${this.title} is ${this.fileSize} MB". Create an instance of EBook and test both parent and child methods

💡 Hint: Use class ClassName { } to define a class. The constructor() method runs when creating instances. Use this.property to create and access properties. Methods are defined inside the class without the function keyword. Use extends to create a child class and super() to call the parent constructor. Create instances with new ClassName(). Use console.log() to verify your methods work!
